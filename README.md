# dotfiles

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/), targeting CachyOS + niri (Wayland) with the noctalia shell.

## Bootstrap a fresh machine

```fish
# 1. Install chezmoi
paru -S chezmoi

# 2. Pull this repo and apply
chezmoi init --apply git@github.com:souchontheo/dotfiles.git
```

`chezmoi apply` will:
- Symlink/write tracked configs to `~/.config/`, `~/.gitconfig`, etc.
- Run `run_once_install-packages.sh` to install all my apps via paru
- Run the `modify_config.kdl` script to inject user includes into the niri config

## Stack

- **Compositor**: niri (Wayland, scrolling tiling)
- **Shell**: noctalia (quickshell-based)
- **Terminal**: ghostty (Maple Mono NF)
- **Shell**: fish (cachyos preset + custom env)
- **Browser**: brave
- **Editor**: nvim (LazyVim)
- **Comms**: vesktop, signal-desktop

## niri config strategy

The CachyOS niri preset (`cachyos-niri-noctalia`) ships configs to `~/.config/niri/cfg/*.kdl`. Rather than editing those files (and losing upstream updates), I keep `cfg/` 100% pristine and put all my overrides in `user_cfg/`.

The two files are wired together via `~/.config/niri/config.kdl`, which:
1. Includes `cfg/*.kdl` (vanilla CachyOS — gets package updates)
2. Includes `user_cfg/*.kdl` afterwards (my stuff — last include wins)
3. Includes `noctalia.kdl` (generated by noctalia at runtime)

The user_cfg includes are appended to `config.kdl` by `modify_config.kdl`, a chezmoi modify script that runs on every `chezmoi apply`. This keeps `config.kdl` itself almost vanilla — only the marked block at the end is managed.

## What's tracked vs not

**Tracked** (anything I created or modified that isn't pure default):
- All of `dot_config/fish/config.fish`, `ghostty/`, `lazygit/`, `nvim/`
- `niri/user_cfg/` and `niri/modify_config.kdl`
- `.gitconfig`, `.gitignore`

**Not tracked** (vanilla / app-generated / regenerable):
- `~/.config/niri/cfg/*` — CachyOS defaults, untouched
- `~/.config/niri/noctalia.kdl` — generated by noctalia at runtime
- `~/.config/noctalia/*.json` — runtime state from the noctalia GUI
- App caches/profiles: `~/.config/{discord,vesktop,BraveSoftware,mozilla,Signal,...}`

## Common ops

```fish
chezmoi diff                 # show pending changes (source vs target)
chezmoi apply                # write source → target
chezmoi edit <file>          # edit source then apply
chezmoi add <file>           # start tracking a file
chezmoi cd                   # cd into the source repo
```
